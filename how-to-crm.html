How to Use the CRM

The CRM allows you to manage all your client relationships in one place. From inviting new clients to viewing booking history and notes, everything you need is available in a single, easy-to-use dashboard.

Accessing the CRM

From your dashboard, click CRM in the left-hand menu. This will open your full client list, where you can search, filter, invite, or manage clients.

  • View all clients in one list
  • Search by name or email
  • Filter and download client data

Inviting a New Client

You can manually invite a new client directly from the CRM. This is ideal when adding individual clients one at a time.

  1. Click the Invite Client button
  2. Enter the client’s first name and last name
  3. Add their email address and phone number
  4. Optionally add notes for internal reference
  5. Click Add to send the invitation

Once invited, the client will appear in your CRM and can begin booking immediately once they accept.

Adding Clients Using a CSV File

If you are importing multiple clients at once, you can upload them using a CSV file. This is useful when migrating from another system.

  1. Click Add Client With CSV
  2. Select a CSV file from your computer
  3. Ensure the file uses the required column headers exactly
  4. Upload the file to add clients in bulk

Your CSV file must use the following headers:

  • first_name
  • last_name
  • email
  • mobile_number
  • notes
  • send_invitation_email

Any missing or extra columns will cause the upload to be rejected. You can download a sample CSV from the upload screen for reference.

Viewing Client Details

Click on any client name in the CRM list to open their individual profile. This gives you a full overview of their activity and information.

  • Client name and unique ID
  • Joining date
  • Internal notes
  • Associated profiles
  • Financial snapshot

You can also switch between tabs to view booking history and messages linked to that client.

Editing Client Information

Client details can be updated at any time. This is useful if contact details change or you want to add notes.

  1. Open the client’s profile
  2. Click Edit Profile
  3. Update name, email, phone number, or notes
  4. Click Update to save changes

All changes are applied instantly and reflected across the system.

Viewing Booking History and Messages

Within each client profile, you can review:

  • Booking history for sessions, classes, or courses
  • Messages exchanged with the client

This gives you full visibility of client activity and communication in one place, helping you deliver a more personal service.
