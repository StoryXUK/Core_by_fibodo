How To Guide

Vouchers

Create gift vouchers, discount packages, and multi-booking offers.

1. Open Vouchers

Where to go:
Use the left-hand menu and click Vouchers.

What you'll see:

  • A list of all vouchers you've created
  • Status indicators (e.g. Active)
  • Search, download, and filter options
  • A Create New Voucher button

2. Create a new voucher

To create a new voucher:

  1. Click Create New Voucher.
  2. The voucher setup wizard will open.
  3. Follow the steps to define how the voucher works.

3. Step 1 – Add voucher information

Start by defining the type of voucher and its basic details.

Voucher types include:

  • Gift Voucher
  • Discount Package
  • Multi Booking

In this step you can:

  • Select the voucher type
  • Add an optional description (up to 200 characters)

4. Step 2 – Select products

Choose which products or sessions the voucher can be applied to.

  1. Select one or more products from the list.
  2. Only selected products will accept this voucher.
  3. Click Next to continue.

Examples include:

  • 1-to-1 sessions
  • Group classes
  • Courses or events

5. Step 3 – Set terms & conditions

Define how the voucher behaves once purchased.

Options include:

  • Whether the voucher has an expiry date
  • If the voucher is transferable
  • Whether it can be purchased as a gift
  • If it can be combined with other offers

Once complete, click Create to activate the voucher.

6. View and manage existing vouchers

All created vouchers appear in the vouchers list.

From here you can:

  • See voucher type, title, creation date, and status
  • Search for a specific voucher
  • Download voucher data
  • Filter vouchers by status or type

7. Edit a voucher after creation

Once a voucher has been created, it's easy to make changes.

  1. Find the voucher in the list.
  2. Click the three dots menu on the right-hand side.
  3. Select the action you want (edit, disable, or manage).

Tip: This allows you to quickly update offers without creating a new voucher from scratch.

8. Voucher best practice

  • Use gift vouchers for seasonal promotions and referrals
  • Apply expiry dates to encourage faster redemption
  • Limit products where needed to protect margins
  • Review voucher performance regularly

Result:
Vouchers give you a flexible way to increase bookings, reward loyalty, and promote your services — all managed from one simple dashboard.
